### clang-petlje

setup:
```
sudo apt-get update
sudo apt-get install libclang-6.0-dev llvm-6.0-dev clang-format
```

cmake:
```
cd build
cmake ..
make
```

test:
```
./petlje <stari> <novi> <do|while|for>
./petlje ulaz.c izlaz.c do
```
